# Audio Files

## Lullabies (`lullabies/`)

The MP3 files in `lullabies/` were generated using [Suno AI](https://suno.com) for personal, non-commercial use as part of a portfolio project. No monetization is involved.

These files are **not licensed for commercial use**. If you fork this project for commercial purposes, replace them with appropriately licensed audio.

## Background Music (`calm.mp3`, `magical.mp3`, etc.)

Mood-based background music tracks for story narration. Place royalty-free MP3s here:

- `calm.mp3` — soft, gentle ambient
- `magical.mp3` — twinkly, ethereal
- `exciting.mp3` — upbeat, adventurous
- `tense.mp3` — suspenseful, dramatic

Recommended free sources:
- https://freesound.org (Creative Commons)
- https://incompetech.com (Kevin MacLeod — CC BY)
- https://pixabay.com/music/

The app gracefully skips music if files are not present.
